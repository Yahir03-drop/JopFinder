📧 Plantillas de Email para JobFinder

🔧 Instrucciones de Configuración

  1. Crear cuenta gratuita en EmailJS.com
  2. Configurar un servicio de email (Gmail, Outlook, etc.)
  3. Crear estas plantillas en EmailJS
  4. Copiar las claves en email-service.js

⚙️ Configuración en email-service.js

Después de crear las plantillas en EmailJS, reemplaza estas líneas en email-service.js:

// Reemplazar con tus datos reales de EmailJS this.emailjsPublicKey = 'tu_public_key_aqui'; this.serviceId = 'tu_service_id'; this.templateId = 'template_welcome'; // Para email de bienvenida

🔑 Dónde encontrar las claves:

  • Public Key: EmailJS Dashboard > Account > API Keys
  • Service ID: EmailJS Dashboard > Email Services
  • Template ID: EmailJS Dashboard > Email Templates

🧪 Probar la Configuración

Una vez configurado, puedes probar desde la consola del navegador:

// Probar envío de email de bienvenida emailService.sendWelcomeEmail({ name: 'Tu Nombre', email: 'tu_email@ejemplo.com', userType: 'candidate' });
